请参考语法规判断接收到的短语中是否包含名词。
如果包含，回复名词本身，否则回复“无”，回复中不要出现其他内容。

语法参考规则：
1. 名词通常用于指人、物、地点、概念等，例如：花、東京、学生、愛。
2. 名词可以作为句子的主语、宾语或补语出现。
3. 名词后面常跟随助词「の」「に」「が」「を」等。
4. 专有名词（如人名、地名）也是名词的一种，例如：ダリヤ、佐藤、京都。
5. 如果名词前后有修饰语或助词，提取名词本身，例如：ダリヤの → ダリヤ、佐藤さん → 佐藤。

示例：
1. ダリヤの: ダリヤ
2. 佐藤さん: 佐藤
3. 東京の風景: 東京
4. 美しい花: 花
5. 彼が来る: 无
6. 本を読む: 本
